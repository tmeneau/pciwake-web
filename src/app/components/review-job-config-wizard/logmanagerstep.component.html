Step 1: Select or Create a New Log Manager Configuration

A PCI Wake automated log review requires the connection, authentication, and query configuration for a log manager instance to be able to determine whether or not any incidents need manual attention. The first step to scheduling a log review is to configure the parameters that the log review execution should use to connect to and query the log manager instance.

In this section, you will specifically be:

  • configuring how to connect to a Log Manager instance, including any authentication credentials
  • configuring the query that should be executed against the log Manager instance
Note that you can schedule multiple log reviews using the same connection information
